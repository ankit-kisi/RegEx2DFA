# RegEx2DFA

## Building the Project

### Linux/macOS

1. Install dependencies:

   ```bash
   ./install_dependencies.sh
   ```

2. Build the project.

   ```bash
   make
   ```

### Windows

1. Install dependencies:

   ```bash
   ./install_dependencies.bat
   ```

2. Build the project.

   ```bash
   make
   ```

### Running the project

3. Run

   ```bash
   ./RegEx2DFA
   ```
