## Installation

### Using Docker

1. Clone the repository:

   ```bash
   git clone https://github.com/ankit-kisi/RegEx2DFA.git
   cd RegEx2DFA
   ```

2. Build the Docker image:

   ```bash
   docker build -t regex2dfa .
   ```

3. Run the Docker container:
   ```bash
   docker run --rm regex2dfa
   ```
